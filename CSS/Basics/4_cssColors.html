CSS Selectors

Numeric Colors

Hex colors

Syntax : h1{ color: #b7150; }
Hex is a shorthand syntax for RGB.

RGB (Red, Green, Blue)

Example : h1 { color: rgb(183, 21, 64); }

RGBA

RGBA stands for red, green, blue alpha. It is an extension of RGB with an additional parameter - alpha which specifies the opacity of the color. Alpha ranges between 0.0(fully transparent) to 1.0(fully opaque).
syntax : rgba(red, green, blue, alpha)

HSL (Hue, Saturation, Lightness)

  • Hue Hue specifies the degree on the color wheel in the range 0 to 360, where 0 means red, 120 means green and 240 means blue

  • Saturation The value of saturation is defined in percentage. 100% saturation means fully saturated, i.e. no grey shades, 50% saturation represents 50% grey, and 0% saturation means fully unsaturated, which is completely grey.

  • Lightness -> It defines the amount of light to be added to the color. 0% no light (black) 50% represents neither dark nor light 100% represents full lightness (white) Example : Syntax: color:hsl(hue, saturation, lightness)
